  • Vigenère Cipher Project:

     A website designed to encrypt and decrypt messages using the Vigenère Cipher. Text can be encrypted with a key and decrypted if the key is known. If there is text encrypted with the Vigenère Cipher where the key is not known, the webpage can cryptanalyze the text to provide the original plaintext and key.

  • Sylver Coinage Project:

     A website designed for playing and testing the Sylver Coinage game. The current version allows you to play another human locally, or a bot locally. Eventually, I hope to implement functionality for testing and writing your own bots, and to play against other humans over the web.

  • Numerical Semigroup Python Package:

     I created a package for working with numerical semigroups in Python. There is currently an existing package that uses SageMath, GAP, and Python to generate numerical semigroups and examine their properties. I found that I often needed to do semigroup math in pure Python so I decided to create this package. Also, I expanded on the traditional definition of a numerical semigroup which requires a finite complement, to look at all numerical monoids regardless of whether they are cofinite or not.

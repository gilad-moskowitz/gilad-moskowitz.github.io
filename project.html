  • Vigenère Cipher Project: A website designed to encrypt and decrypt messages using the Vigenère Cipher. Text can be encrypted with a key and decrypted if the key is known. If there is text encrypted with the Vigenère Cipher where the key is not known, the webpage can cryptanalyze the text to provide the original plaintext and key.

Description

  This program uses the Vigenère Cipher to encode and decode English text. When encoding English text, simply write the text you want encoded in the "plaintext" textbox, and the key in the corresponding "key" textbox. You can then press on "Encode (All)" which will return ciphertext that includes all the non-letter characters into the "ciphertext" textbox. Alternatively, you can press on "Encode (Only letters)" which will return ciphertext with only the letter characters.

  You can also decode an English text encoded with the Vigenère Cipher by inputting it in the "ciphertext" textbox. The algorithm for this process is described here. The decoder can only work effeciently for large sized ciphertext, but the algorithm works well enough that it can often decipher text of "length" at least 30. In this instance, the "length" of the ciphertext refers to the number of English letters contained in the text.

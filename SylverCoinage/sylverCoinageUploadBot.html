Sylver Coinage

Here you have the option to upload a bot and test it against my current bot. Note, when you upload the bot, please give the page about 30 seconds to run some tests using your bot and make sure it works as intented. The rules for your code are as follows:
  • Your function takes in two arguments, both lists. The first is the list of all the moves already played. The second is a list of all remaining legal moves if there are only finitely many of them.
  • Your function returns an integer. If it is the first move of the game, your function returns an integer less than 30. In a given infinite position, your funtion returns a move less than 100. If the position is finite, your bot can return any move.
  • The function I will call to get your bot move is going to be: myNewBot(movesAlready, remainingMoves).
